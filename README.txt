MINDSTREAM
==============

Mindstream is a collection of python modules and applications that are closely related to the Moments module, but beyond the scope of that module.  This collection is the foundation for my work flow to process tasks, and includes:  
  
mindstream
  -python modules to:
    -launch common classes of processes on the local system
    -move and sort media from different devices (import_media.py)

editor
  -stand alone wx based moments editor. 
   see also moments/editors for standard editor configurations

pose
  -web server to load and browse a collection of moments based journals

scripts
  -utilities for processing moments based journals

moments-web
  -an example of a Google App Engine based application 
   to collect moment entries via the web.  

tagger
  -wx based desktop application to manually tag many moment entries
